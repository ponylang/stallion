# lori_http_server

An HTTP server for [Pony](https://www.ponylang.io/).

## Status

This library is under active development. The API is not yet stable.

## Installation

* Install [corral](https://github.com/ponylang/corral)
* `corral add github.com/ponylang/lori_http_server.git --version 0.0.0`
* `corral fetch` to fetch your dependencies
* `use "http_server"` to include this package
* `corral run -- ponyc` to compile your application
